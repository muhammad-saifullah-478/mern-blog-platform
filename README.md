# AI Blog Platform

A modern full-stack blog platform that allows administrators to manage content efficiently while providing users with an engaging reading experience. The platform includes an Admin Dashboard, AI Chatbot integration, Trending Posts, and a responsive user interface.

## 🚀 Features

### User Features
- Browse and read blog posts
- View trending posts
- Responsive design for desktop, tablet, and mobile
- Fast and user-friendly interface
- AI Chatbot for user assistance

### Admin Features
- Secure Admin Dashboard
- Create new blog posts
- Edit existing posts
- Delete posts
- Manage content in real time

## 🛠️ Tech Stack

### Frontend
- React.js / Next.js
- Tailwind CSS
- Redux Toolkit

### Backend
- Node.js
- Express.js

### Database
- MongoDB

### Authentication
- JWT Authentication
